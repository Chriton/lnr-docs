# isNormalizedName

This function checks if a domain string is normalized.

```
function isNormalizedName(_name: string): boolean
```

## Usage

```typescript jsx title="MyComponent.tsx"
import { lnr } from '@linagee/lnr-ethers-react';

const MyComponent = () => {
    const domainString = "0xhal.og";

  return (
    <div>
      <h1>{lnr.utils.isNormalizedName(domainString) ? "Normalized" : "Not Normalized"}</h1>
    </div>
  );
};

export default MyComponent;
```

The above code will render the following:

```
Normalized
```

## Parameters

- **_name** (string): A domain string to be checked for normalization.

## Returns

- **(boolean)**: True if the domain string is normalized, false otherwise.
